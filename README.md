# 网页基本功能展示项目

这是一个用于学习计算机网络应用层的静态网页项目，展示了HTML的基本功能。

## 项目结构

```
lab03/
├── index.html          # 主页，展示所有基本功能
├── page2.html          # 第二页，演示页面跳转
├── css/
│   └── style.css       # 自定义样式文件
├── images/             # 图片文件夹（需要自己添加图片）
└── media/              # 音视频文件夹（需要自己添加媒体文件）
```

## 使用说明

### 1. 直接打开
双击 `index.html` 文件即可在浏览器中打开。

### 2. 使用本地服务器（推荐）
```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

### 3. 添加媒体文件

#### 添加图片
将你的图片文件放入 `images/` 文件夹，然后修改 `index.html` 中的图片路径：
```html
<img src="images/your-image.jpg" alt="描述">
```

#### 添加音频
将音频文件（如 .mp3, .ogg）放入 `media/` 文件夹，音频播放器会自动识别。

#### 添加视频
将视频文件（如 .mp4, .webm）放入 `media/` 文件夹，视频播放器会自动识别。

## 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式美化
- **Bootstrap 5**: UI框架（通过CDN引入）
- **Bootstrap Icons**: 图标库

## 特点

- ✨ 响应式设计，适配手机、平板、电脑
- 🎨 使用 Bootstrap 5 框架，界面美观
- 📱 移动端友好
- 🚀 纯静态页面，无需后端
- 💡 代码注释详细，便于学习

## 学习要点

1. **HTTP协议**: 理解浏览器如何请求和接收网页资源
2. **URL结构**: 了解相对路径和绝对路径
3. **HTML标签**: 掌握常用HTML标签的使用
4. **多媒体**: 学习如何在网页中嵌入图片、音频、视频
5. **页面导航**: 理解页面间的跳转机制

## 注意事项

- 音视频文件需要自己准备并放入 `media/` 文件夹
- 如果要使用本地图片，请放入 `images/` 文件夹
- 建议使用本地服务器运行，避免某些浏览器的跨域限制
